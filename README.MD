# BlackJack Vite


Pasos para ejecutar este proyecto:

1. Clonar repositorio
2. Ejecutar ```npm install``` para reconstruir los modulos de note
3. Correr el devServer ```npm run dev```
4. Abir el proyecto en: ```http://localhost:5173```
